Principles

  • Data is interchangeable between charts
  • Minimal JS needed for chart to function
  • Add as much or as little html as you want to create the chart
  • Re-usable component
  • CSS Vars to customise
  • Interaction to show/hide datasets and even update the chart type
  • Live charts - auto update after table is updated
