Principles

Basic concepts and rules to help guide us to making the correct choices.

  • Data is interchangeable between charts
  • Minimal JS needed for chart to function
  • Add as much or as little html as you want to create the chart (JS functions can populate the bits missing)
  • CSS Vars can be used to customise the chart (Sass vars can also be used)
  • Interaction to show/hide datasets
  • Charts update when the data is changed; allowing for live charts
  • Semantic structure and accessible by default
